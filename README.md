### 🔷 Python (Data Processing & Analysis)
Python was used for data cleaning, transformation, and exploratory analysis:
- Data preprocessing and cleaning
- Handling missing values
- Feature engineering (age grouping, frequency mapping)
- Exploratory Data Analysis (EDA)

---

### 🟦 SQL Server (Data Storage & Business Analysis)
SQL Server was used as the analytical database layer:
- Structured data storage
- Data modeling and transformation
- Stored procedures for automated analysis
- Business-driven SQL queries (KPIs, segmentation, revenue analysis)

---

### 🟣 Power BI (Data Visualization & Reporting)
Power BI was used to build an interactive business dashboard:
- KPI cards (Revenue, Customers, Average Spend)
- Customer segmentation analysis
- Product performance insights
- Interactive filters and slicers

---

## 🔄 End-to-End Data Pipeline

CSV Dataset → Python (Cleaning & Feature Engineering) → SQL Server (Storage & Analysis) → Power BI (Dashboard & Insights)
