## 🛠️ Tech Stack

This project is an end-to-end Customer Loyalty Analytics solution that transforms raw customer data into actionable business insights through a structured analytics pipeline.

### 🧰 Tools & Technologies
🐍 Python | 🗄️ SQL Server | 📊 Power BI | 📁 CSV Dataset

---

### 🔷 Python (Data Processing & Analysis)
Python was used for data cleaning, transformation, and exploratory analysis:
- Data preprocessing and cleaning
- Handling missing values
- Feature engineering (age grouping, frequency mapping)
- Exploratory Data Analysis (EDA)

---

### 🟦 SQL Server (Data Storage & Business Analysis)
SQL Server was used as the analytical database layer:
- Structured data storage
- Data modeling and transformation
- Stored procedures for automated analysis
- Business-driven SQL queries (KPIs, segmentation, revenue analysis)

---

### 🟣 Power BI (Data Visualization & Reporting)
Power BI was used to build an interactive business dashboard:
- KPI cards (Revenue, Customers, Average Spend)
- Customer segmentation analysis
- Product performance insights
- Interactive filters and slicers

---

## 🔄 End-to-End Data Pipeline

CSV Dataset → Python (Cleaning & Feature Engineering) → SQL Server (Storage & Analysis) → Power BI (Dashboard & Insights)

---

## 🏗️ Architecture Overview

📁 Raw Dataset (CSV)  
⬇  
🐍 Python Layer (Cleaning & Feature Engineering)  
⬇  
🗄️ SQL Server Layer (Storage + Stored Procedures + Analysis)  
⬇  
📊 Power BI Layer (Visualization + Business Insights)

---

## 🎯 Key Highlights

- End-to-end real-world data analytics pipeline  
- Strong focus on customer behavior and revenue analysis  
- Automated SQL stored procedure for reusable insights  
- Interactive and business-focused Power BI dashboard
